# App assets

Drop the app's real icons here once you have them designed:

- `images/icon.png` — 1024×1024 app icon.
- `images/adaptive-icon.png` — 1024×1024 foreground layer for Android.
- `images/splash-icon.png` — splash screen artwork.
- `images/favicon.png` — 48×48 favicon for the web build.

When the files exist, wire them into `apps/mobile/app.json` under the `icon`, `android.adaptiveIcon`, `splash`, and `web.favicon` fields. Expo uses sensible defaults until then.
